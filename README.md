SharePoint OnlineのフォルダページでフォルダURLとファイル一覧をまとめてコピーするブックマークレットです。

## できること

SharePointのフォルダページでフォルダ名、URL、ファイル一覧を一度にコピーできます。TeamsやOutlookに貼り付けるとリンクや装飾が保持されます。

8種類の形式から用途に応じて選択可能で、サイト名からフォルダまでの階層構造も表示されます。

## コピー形式

### リンク付きコピー（Teams/Outlook向け）
- **全部盛り（シンプル）** - フォルダ情報 + ファイル一覧 + 階層パス
- **全部盛り（リッチ）** - 上記 + ファイル種類アイコン付き
- **フォルダ（リンク）のみ** - フォルダ名とURLのみ
- **フォルダ+アイテム（シンプル）** - フォルダ情報 + ファイル一覧（デフォルト・Enterキー対応）
- **フォルダ+アイテム（リッチ）** - 上記 + ファイル種類アイコン付き

### テキストコピー
- **アイテム一覧（引用）** - ファイル一覧を引用形式で
- **アイテム一覧（箇条書き）** - ファイル一覧を箇条書き形式で  
- **アイテム一覧** - ファイル名のみシンプルに
- **詳細パス（Tree）** - 階層構造をツリー形式で
- **詳細パス（パンくず）** - 階層構造をパンくず形式で

## インストール方法

### Chrome

1. ブックマークバーを表示（Chrome右上「⋮」→「ブックマーク」→「ブックマークバーを表示」）

2. ブックマークバーで右クリック→「ページを追加」

3. 名前に「SharePoint コピー」、URLに以下のコードを貼り付け
   ```javascript
   [ここにブックマークレットのコードが入ります]
   ```

4. 保存

### Microsoft Edge

1. お気に入りバーを表示（Edge右上「⋯」→「お気に入り」→「お気に入りバーを表示」）

2. お気に入りバーで右クリック→「お気に入りを追加」

3. 名前に「SharePoint コピー」、URLに以下のコードを貼り付け
   ```javascript
   [ここにブックマークレットのコードが入ります]
   ```

4. 保存

## 使い方

1. SharePointのフォルダページを開く

2. 特定のファイルのみ対象にしたい場合はチェックボックスで選択（何も選択しない場合は全ファイルが対象）

3. ブックマークバーの「SharePoint コピー」をクリック

4. 表示されたダイアログから用途に応じてボタンをクリック（Enterキーで「フォルダ+アイテム（シンプル）」が実行）

5. Teams、Outlook、メモ帳などに貼り付け

## トラブルシューティング

### 🚨 重要：コピーしたリンクが展開されてしまう場合

**Teamsで貼り付け後に Ctrl+Z を押すとリンク展開を取り消せます**

これはTeamsの仕様で、URLを含むテキストを貼り付けると自動的にプレビューが展開されます。Ctrl+Zで元のテキスト形式に戻せるので覚えておいてください。

### その他のよくある問題

**ブックマークレットが動かない**
- ページが完全に読み込まれてから実行
- ブラウザのページを更新してから再試行

**ファイル一覧が表示されない** 
- SharePointでファイルを選択するか、リスト表示にしてから実行

**階層パスが正しく表示されない**
- ブレッドクラム（パンくずリスト）が表示されている状態で実行

## セキュリティ

このブックマークレットは外部サーバーと通信しません。すべての処理はブラウザ内で完結し、SharePointの認証情報やファイル内容を外部に送信することはありません。
